Paths

/cumulative_returns

  • POST

Summary

Allows users to calculate the cumulative return of the event study.

Description

This endpoint returns the cumulative returns (%) to determine whether a certain event has affected the stock price abnormally in some way. The response includes the display name and other details about each product, and lists the products in the proper display order.

Query parameters

  • upper_windowR

    The upper window of time relative to the event date.

  • lower_windowR

    The lower window of time relative to the event date.

  • upper_<var_name>R

    The upper threshold value for the event type.

  • lower_<var_name>R

    The lower threshold value for the event type.

  • stock_price_fileasdfasdfs R

    The .csv file containing the stock prices.

  • stock_characteristic fileR

    The .csv file containing the stock characteristic.
