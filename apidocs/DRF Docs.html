Endpoint

/cumulative_returns

  • POST

Summary

Allows users to calculate the cumulative return of the event study.

Description

This endpoint returns the cumulative returns (%) to determine whether a certain event has affected the stock price abnormally in some way. The response includes the display name and other details about each product, and lists the products in the proper display order.

Query parameters

  • upper_windowR

    Query Parameter: The upper window of time relative to the event date.

  • lower_windowR

    Query Parameter: The lower window of time relative to the event date.

  • upper_<var_name>R

    Query Parameter: The upper threshold value for the event type.

  • lower_<var_name>R

    Query Parameter: The lower threshold value for the event type.

  • stock_price_fileR

    Body Parameter: The .csv file containing the stock prices.

  • stock_characteristic fileR

    Body Parameter: The .csv file containing the stock characteristic.

Return type

  • Cumulative ReturnsD

    JSON array object: The cumulative returns, according to the given parameters.

Produces

This API call produces the following media types according to the Accept request header; the media type will be conveyed by the Content-Type response header:
  • application/json

Responses

  • 200

    All good, returns a JSON array.

  • 404

    The provided parameters are incorrect.

  • 422

    The file(s) provided are not in CSV format.
