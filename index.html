Event Study API

Event study nade by prOve_it which analyses events and determines whether the events have an affect on stocks.

URI related to this API: http://prove-it-unsw.herokuapp.com/api/v1/event-study
Version: 1.0

  1. post /cumulative_return
post /cumulative_return
Retrun cumulative return of event study. (cumulativeReturnPost)
This endpoint returns the cumulative returns (%) to determine whether \na certain event has affected the stock price abnormally in some way. The \nresponse includes the display name and other details about each product, \nand lists the products in the proper display order.\n

Query parameters

upper_window (required)
Query Parameter — The upper window of time relative to the event date.
lower_window (required)
Query Parameter — Lower window of time relative to the event date.
upper_<var_name> (required)
Query Parameter — Upper number for the event type.
lower_<var_name> (required)
Query Parameter — Lower number for the event type.
stock_price_file (required)
Body Parameter — Stock price file.
stock_chara_file (required)
Body Parameter — Stock characteristic file.

Return type

array[completeJSON]

Produces

This API call produces the following media types according to the Accept request header; the media type will be conveyed by the Content-Type response header.
  • application/json

Responses

  • 200

    A JSON array.

  • 404

    Provided parameters are incorrect.

  • 422

    File(s) are not in CSV format.
